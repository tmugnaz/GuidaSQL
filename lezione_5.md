# Comandi
### Funzioni di aggregazione
- **AVG** -> media
- **SUM** -> somma
- **MIN** -> valore minimo
- **MAX** -> valore massimo
- **COUNT([DISTINCT])** -> calcolo il numero degli elementi(con o senza duplicati), es: SELECT COUNT(DISTINCT CODFID) as "Numero clienti" FROM ...
- **COUNT(*)** -> calcolo il numero totale delle righe
